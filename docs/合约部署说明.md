# ConfChainCore 合约部署说明

## 合约概览

| 文件 | `contracts/src/ConfChainCore.sol` |
|------|------------------------------------|
| 函数 | `submitCopyright` / `submitReview` / `finalizeDecision` / 3 个 view 函数 |
| 链 | FISCO BCOS 3.11 Air |
| 部署账号 | WeBASE 管理员账号（即合约 `owner`） |

---

## 方式一：通过 WeBASE-Front 控制台部署（推荐）

### 1. 编译合约

在 WeBASE-Front 页面（`http://Ubuntu-IP:5002`）：

1. 进入 **合约管理 → 合约 IDE**
2. 新建文件，粘贴 `contracts/src/ConfChainCore.sol` 全文
3. 点击 **编译**，确认无错误

### 2. 部署合约

1. 在合约 IDE 中点击 **部署**
2. 选择一个已有私钥账号（将成为合约 `owner`）
3. 点击确认，等待交易回执
4. 复制部署成功后显示的 **合约地址**（如 `0x1234...abcd`）

### 3. 更新环境变量

在 Windows 开发环境的 `.env` 和 `apps/api/.env` 中更新：

```env
FISCO_CONTRACT_COPYRIGHT=0x你的合约地址
FISCO_CONTRACT_REVIEW=0x你的合约地址
```

> 注意：版权存证与审稿评审目前共用同一个 `ConfChainCore` 合约，两个变量填相同的合约地址。

重启后端：
```bash
pnpm --filter @confchain/api dev
```

---

## 方式二：通过 Hardhat 脚本部署

> 适用于本地测试或需要自动化部署场景。

### 1. 配置 hardhat.config.ts

编辑 `contracts/hardhat.config.ts`，添加 FISCO 网络（通过 JSON-RPC）：

```typescript
networks: {
  fisco: {
    url: process.env.FISCO_RPC_URL || "http://192.168.229.130:20200",
    accounts: ["0x你的私钥"],
    chainId: 20200,
  },
}
```

### 2. 部署脚本

```bash
cd contracts
npx hardhat run scripts/deploy.ts --network fisco
```

脚本会输出合约地址，填入 `.env`。

---

## 验证部署

部署成功后，可通过 WeBASE-Front 合约调用界面调用 `owner()` 函数，确认返回的地址与部署账号一致。

---

## 合约接口速查

| 函数 | 参数 | 调用方 | 说明 |
|------|------|--------|------|
| `submitCopyright` | fileHash, author, timestamp, metadataHash | onlyOwner | 版权存证 |
| `submitReview` | paperId, reviewer, score, recommendation, commentHash | onlyOwner | 审稿上链 |
| `finalizeDecision` | paperId, decision | onlyOwner | 裁定上链 |
| `getCopyright` | fileHash | public | 查询存证 |
| `getReviewCount` | paperId | public | 查询审稿数量 |
| `getReview` | paperId, index | public | 查询单条审稿 |
| `getDecision` | paperId | public | 查询裁定结论 |

---

## 后端调用流程

```
作者上传文件
  → PapersService.uploadAndCertify()
    → 计算 SHA-256 (fileHash)
    → 计算元数据哈希 (metadataHash)
    → FiscoService.certifyCopyright()
      → POST /WeBASE-Front/trans/handle
        → ConfChainCore.submitCopyright(fileHash, author, ts, metadataHash)
    → 保存 txHash + blockHeight 至 Paper 表
    → 写入 ChainTransaction 表
```
