# ConfChain 实施阶段与交付说明

## Phase 0 基建
- Monorepo 结构、Docker Compose、CI、环境变量模板。
- 交付：可一键启动的基础设施与工程骨架。

## Phase 1 身份与权限
- 实现注册、登录、JWT 鉴权、RBAC。
- 交付：`auth`、`users` 模块与接口。

## Phase 2 版权存证闭环
- 稿件创建、哈希计算、链存证回执记录、哈希验证。
- 交付：`papers` 模块与 `ChainTransaction` 记录逻辑。

## Phase 3 匿名审稿主流程
- 任务分配、评审提交、结果裁定。
- 交付：`reviews` 模块端到端骨架。

## Phase 4 运维与配置
- 节点状态、交易溯源、配置模块预留。
- 交付：`blockchain` 管理接口。

## Phase 5 安全性能
- Helmet/CORS/ValidationPipe、错误边界和重试机制预留。
- 后续可接入：限流、缓存、压测脚本。

## Phase 6 发布与文档
- README、架构说明、CI、合约脚本与部署入口。
- 交付：可演示的 MVP 主干代码。
